## Shortcode

在内容中使用可复用模板片段。启用后会处理 Page 和 Section 的 `Content`、`Summary`。

```html
<shortcode youtube id="dQw4w9WgXcQ" />
```

```html
<shortcode code lang="python">
print("hello")
</shortcode>
```

完整用法、模板变量、`:::shortcode` 语法与错误回退行为见 [短代码 (Shortcode)](/content/shortcodes/)。
